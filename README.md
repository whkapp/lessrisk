# LessRisk TradeReceiver EA

MetaTrader 5 Expert Advisor — ontvangt signalen van de LessRisk server en toont een trade-panel.

## Versies
- **v3.02** — Dynamische breedte (vult chart), BGR kleuren fix (SELL=rood, BUY=groen), 0.10 lot default, layout overlaps opgelost, status-box tekst spacing fix

## Installatie
1. Open MetaEditor in MT5 (F4)
2. Open `LRNG_TradeReceiver.mq5`
3. Compileer (F7)
4. Sleep de EA naar een chart
