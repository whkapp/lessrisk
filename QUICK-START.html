MT5 Quick Start

LRNG_TradeReceiver v3.01 — direct server polling, no Helper needed.

Installation

  1. 1
    Run LessRisk_Installer.exe — EA copied to MT5 Experts folder automatically.
  2. 2
    MT5: Tools > Options > Expert Advisors — enable WebRequest for URL below.
  3. 3
    Navigator: drag LRNG_TradeReceiver onto any chart (e.g. BTCUSD H1).
  4. 4
    EA settings > Common: enable Allow Algo Trading, click OK.
  5. 5
    Enable Algo Trading in MT5 toolbar (green). Smiley appears top-right.
